技术发展规划

发表于 2017-12-03 | 分类于 成长

为什么会走到现在

从高一开始，自己就想好了大学要报考的专业————计算机相关专业。经历了不想多说的高考进入到大学以后，觉得顺风顺水，自己在 C 语言开课之前就已经读完了一本 C 语言程序设计。在最后的课程设计中，用SDL库做一个自己照片的拼图小游戏，得了不错的分数，顿时成就感爆棚。之后一直在专业课上表现不错。

读到这里，按照剧情的发展，后面毕业以后考到某某大学读研继续深造或者进入某某大公司开始实习。然而现实并不是这样，先是大三新浪面试失败，然后大四考研失利。最后，随便找了一家公司开始实习工作了。。。

最近偶然在二叉树视频里看到了鸟哥的视频，研究生被调剂到计算机专业，之后投递雅虎中国的实习生失败了。但是鸟哥并没有放弃，而是看到不足以后努力弥补，时隔半年成功进入了雅虎。这里之所以提到鸟哥，是因为当时新浪面试的时候，简历是发到鸟哥邮箱里的（当时并不知道鸟哥是如此的niubility）!面试时面试题做得一般，面试官问了一个红黑树的概念，自己一脸懵逼，自信心被打垮。

如今，时隔三年，红黑树是什么，还不是很明白。暂且不说看到不足以后的处理方式，自己计算机感“兴趣”，这是停留在专业选择上，并没有什么实质性的了解和实践。除了理论知识经过记忆周期遗忘以外，还有解决问题的不求甚解和缺乏对计算机世界的好奇心。这种所谓的兴趣和表面上的努力让自己走到了今天。还有一个原因是心态，这个虽然异常重要，但是涉及到玄学，就不多说了。

急诊科医生里的男主角说的好，当不了一个救死扶伤的神，也不要做一个普通人。自己花了三年时间，看到了差距，接下来是时候该做点什么了。
